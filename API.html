RTCStarClient

PeerJS Key .key

PeerJS Host (if you your own PeerJS server) .setHost(hostUrl)

Start the client .start()

Stop the client .stop()

Send a request object to server, example's type is 'Chat' .send({type:'Chat', data:{....}})

Broadcast a message object to all clients, example's type is 'Chat' .broadcast({type:'Chat', data:{....}})

Call a peer .call(peerId, mediastream)

Listen for client events (Multiple listeners are allowed):
[event, callback args]
'Open': (clientPeerId)
'Close'
'Error': (err)
'ClientEnter': (peerId)
'ClientLeave': (peerId)
'ClientList': ([peerId, peerId, peerId...])
'Call': (callConnPeerJS)
.onClientEvent(event, function(args){

})

Remove listener for client events .removeOnClientEvent(event, function(){

})

Listen for server messages (Multiple listeners to each type are allowed) .onMessage(type, function(message){

})

Remove listener for server messages .removeOnMessage(type, function(){

})

Set debug mode to print out log messages. .debug()

Get the client peerId of PeerJS .getClientPeerId()
