ChatWidgetServer

Chat Widget user only require knowledge of chat widget client API



ChatWidgetClient

Broadcast a message to all peers connected to the same rtcstarserver. .sendChat(message)

Handler to handle message that was broadcasted to you .onMessage(function(message){
//message.peerid = sender id
//message.text is the content
})

VideoWidgetServer

Video Widget user only require knowledge of video widget client API.



VideoWidgetClient

Handler for your own stream .onStart(function(localstream){
//handle the stream here
})

Handler when a remote stream come in .oncall(function(stream){
//handle the stream here
})


RTCStarServer

Start the server. .start()

Listen for server id after the server.start() method has been called. .onServerEvent('open',function(server_id){

})

Whenever a new client has connected with you .onServerEvent('ClientEnter',function(peer_id){

})

Sending a connected peer a message .send(peerId, message)

Broadcast message to everyone .broadcast(message)

RTCStarClient

Connect the client to the server .start(server_id)

Sending a message to the server .request(message)
